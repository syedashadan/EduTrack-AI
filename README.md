# 🎓 EduTrack AI - Student Management System

## 📌 Overview
EduTrack AI is a Student Management System built using Flask and SQLite. It helps manage student records with authentication, profile management, photo uploads, statistics, and export features.

## 🚀 Features

- Secure Login System
- Dashboard with Statistics
- Add Student
- View Students
- Edit Student
- Delete Student
- Search Students
- Student Profile
- Upload Student Photo
- Export to Excel
- Export to PDF
- Student Statistics Chart

## 🛠 Technologies Used

- Python
- Flask
- SQLite
- Bootstrap 5
- HTML
- CSS
- JavaScript
- Chart.js
- OpenPyXL
- ReportLab

## ▶️ Run the Project

```bash
pip install -r requirements.txt
python app.py
```
